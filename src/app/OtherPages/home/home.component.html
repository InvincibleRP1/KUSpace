User Features and Guides

  • User must be a student of University of Kalyani
  • Mentioning Batch Year and Course is adviced to interact with fellow classmates
  • Profile page should be updated to share details with your peers.
  • Users can know about the events and Job opportunities through this web app
  • Get result of any semester under the results section
  • Discuss about career opportunities and growth with seniors
  • Get to know about the schedules and events of the department

Faculty Features and Guide

  • Need valid Faculty-id
  • Access to the admin panel where student database, courses and all important issues of department can be addressed.
  • Faculty can contact to each user for any purpose
  • Faculty can check any student details
  • Faculties also need to have a user profile
